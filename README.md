# AI vs Real Image Detector

A PyTorch-based project to classify images as **REAL** or **FAKE** (AI-generated) using a CNN.

## Project Overview
- Loads and preprocesses real and fake images.
- Trains a CNN model.
- Saves model weights for inference.
- Provides a script to predict new images.
